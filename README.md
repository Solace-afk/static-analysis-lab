# Tentang Repository
Repository **static-analysis-lab** ini dibuat sebagai portofolio dan bukti nyata proses belajar Reverse Engineering. Fokus utama dari repositori ini adalah mendokumentasikan analisis statis pada dua *binary* berformat PE (Portable Executable).

Analisis statis adalah metode pembedahan program tanpa mengeksekusinya secara langsung, guna mengidentifikasi struktur file, kapabilitas, dan potensi perilakunya.
